# 📊 Data Analysis 数据分析案例集

> Python 数据分析自学笔记与实践案例，涵盖 Pandas、Matplotlib、数据清洗、可视化与简单建模。

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange.svg)](https://jupyter.org/)

---

## 📖 项目简介

本仓库收录了 **16 个 Jupyter Notebook 数据分析案例**，适合：

- 🔰 Python / 数据分析初学者
- 📚 正在学习 Pandas、Matplotlib 的同学
- 💼 想积累数据分析作品集的朋友

每个 Notebook 都是一个完整的小项目，包含：**数据读取 → 清洗 → 分析 → 可视化 → 结论**。

## 🛠️ 环境要求

```bash
Python >= 3.8
```

主要依赖库：

```bash
pip install pandas matplotlib numpy jupyter seaborn scikit-learn
```

---

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/njq0204/data-analysis.git
cd data-analysis
```

### 2. 启动 Jupyter

```bash
jupyter notebook
```

### 3. 选择一个 Notebook 开始学习

建议新手从以下顺序入手：

```
对比Excel.ipynb → 泰坦尼克号.ipynb → 散点图.ipynb → 线性回归.ipynb
```

## 📚 推荐学习路线

